# 📦 4H Market — Sistema de Control de Mercaderías

Sistema de gestión de inventario en consola desarrollado en Python para el negocio **4H Market**.

## ✅ Funcionalidades

- Ver stock actual de todos los productos
- Agregar productos nuevos con precio y stock mínimo
- Registrar entradas (reposición de mercadería)
- Registrar salidas (ventas o consumo)
- Alertas automáticas cuando el stock baja del mínimo
- Historial completo de movimientos con fecha y hora
- Eliminar productos

## ▶️ Cómo ejecutar

1. Tener Python 3 instalado
2. Ejecutar en la terminal:python inventario_4h.py
## 🛠️ Tecnologías

- Python 3
- Módulos: csv, os, datetime
